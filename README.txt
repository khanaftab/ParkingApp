--------------------------------------------
Parking Application requirement 
--------------------------------------------
    I have tested this application in java 7
  1. Have java in class path.
  2. Please use java 7 or higher.
  
---------------------------------------------------
How to start Parking Application (User Interective)
---------------------------------------------------
  1. Unzip The source file.
  2. Go to Terminal and execute my_program.sh
  
---------------------------------------------------
How to start Parking Application (File as an input)
---------------------------------------------------
  1. Unzip The source file.
  2. Go to Terminal and execute my_program.sh (filePathArgument)
  
You can mannualy compile and run this application using main class ParkingApp.java.
